\begin{Bacteriabox}{\bf {乳酸杆菌属（\emph {Lactobacillus spp.}）}}
\begin{spacing}{1.5}

3）延缓衰老，促进肠道蠕动，促进营养物质吸收，有利于肠道健康。}
