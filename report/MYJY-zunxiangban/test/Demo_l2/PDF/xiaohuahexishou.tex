蛋白质、脂类、碳水化合物是维持人体生命及健康的重要营养物质。蛋白质是生命的物质基础，构成人体各个组织器官，实现各种生理功能；脂类是人体主要储能物质；碳水化合物是机体最重要的直接供能物质。肠道菌群在蛋白质、脂类、碳水化合物的消化、代谢过程中起着重要的作用。

您肠道内参与蛋白降解产物、脂肪降解产物、碳水化合物降解产物的菌群指标存在异常，不利于人体健康，需引起注意，建议持续监测肠道健康。
