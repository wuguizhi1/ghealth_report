您的肠道菌群类别总共有105种，已知细菌103种；未知细菌2种。

您的肠道菌群多样性水平
低，表明您的菌群平衡较差，需要注意肠道健康
（儿童和老年人由于年龄因素，肠道菌群多样性水平比一般成年人差）。

您的细菌敏感性、阿克曼氏菌属、乳杆菌属指标优于参考人群，有利于抵抗疾病、调节免疫、保持正常体重等；您的肠道菌群多样性、有益菌、有害菌等指标存在异常，可能增加菌群失衡的风险，不利于肠道健康、产生有益物质、抑制肠道炎症，需引起注意，建议持续监测肠道健康。
