\vspace*{0.5mm}
\begin{spacing}{1.5}
\noindent\color{black70}\fontsize{9pt}{12pt}\selectfont {{\raisebox{1pt}{\color{black70}$\bullet$}}\, 本次检测提示菌群多样性
正常、
有益菌
偏低、
有害菌
正常，
有益菌含量指标异常，
可能导致肠道环境变差，减重难度增大。建议先调理肠道菌群，恢复肠道微生态平衡，健康的肠道菌群状态有助于减重。
}\\
\noindent\color{black70}\fontsize{9pt}{12pt}\selectfont {{\raisebox{1pt}{\color{black70}$\bullet$}}\,
肠杆菌属、梭菌属、布劳特氏菌属、多尔氏菌属、沙门氏菌属等胖菌指标偏高，
长期胖菌/瘦菌结果异常，可引起肥胖等健康问题，不利于控制体重。
}\\
\noindent\color{black70}\fontsize{9pt}{12pt}\selectfont {{\raisebox{1pt}{\color{black70}$\bullet$}}\,
检测出1种致病菌，其中脆弱拟杆菌含量高于参考人群，需要关注，建议您定期监测，预防致病菌的潜在危害。
}
\end{spacing}
