{\bf\sanhao 粪便状态}
\end{center}

\bigskip

粪便状态分析，指的是在临床实践中，有几个和粪便自身状态相关的指标，这些指标被认为有很高的诊断价
值。我们的检测中包括了临床中常用的二项指标：粪便的质地和粪便的颜色。
\par
您本次粪便健康状况依据布里斯托大便分类法检测，由于单次粪便颜色易受到近期食物的影响（红肉火龙果，西瓜等），仅供参考。若您的粪便形态和颜色长期处于不健康状态，建议及时进行专科检查、诊治。
